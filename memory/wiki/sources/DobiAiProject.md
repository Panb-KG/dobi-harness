## 本地运行

**前置条件:** Node.js

1. 安装依赖:
   ```bash
   npm install
   ```

2. 设置环境变量:
   ```bash
   GEMINI_API_KEY=<your_key>
   ```

3. 运行开发服务器:
   ```bash
   npm run dev
   ```
