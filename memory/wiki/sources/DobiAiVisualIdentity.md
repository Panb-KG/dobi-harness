## Logo & Emoji Matrix

### 核心 Logo

由两个花括号 `{}` 组成，通过改变内部字符让品牌在交互中产生情绪共鸣。

### Emoji 表情矩阵

| Emoji | 含义 | 使用场景 |
|-------|------|----------|
| `{^_^}` | 就绪 / READY | 准备状态 |
| `{;_;}` | 调试 / DEBUG | 遇到问题 |
| `{@_@}` | 思考 / THINKING | 处理中 |
| `{!_!}` | 警报 / ALERT | 警告/错误 |
| `{*_*}` | 自由 / FREE | 完成/释放 |
| `{$_$}` | 收米 / SUCCESS | 成功/收益 |
